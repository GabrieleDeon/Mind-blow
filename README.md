# Jogo Mind Blow
Jogo criado como projeto final da disciplina de Tópicos Especiais Integrados.

Neste jogo você encontrará um padrão de palavras, imagens ou conjuntos numéricos, bem como um conjunto de alternativas, dentre as quais somente uma se enquadra nos padrões de cada questão. Ao final do jogo, o jogador recebe a contagem do tempo que obteve na partida, bem como a sua pontuação de acordo com os seus acertos para que possa visualizar seu desempenho.

## Tecnologias
- JQuery
- Javascript
- HTML
- CSS
